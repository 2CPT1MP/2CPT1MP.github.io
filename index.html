Option Guesser v2.0 Beta

This application was developed by Blade Wolfmoon. Also known as 2CPT1MP. Java programming language was used. However, the application is portable. The runtime environment and installer are included with the bundle so it can run on any computer.

This application can be used to add random options for the application to choose from and then ask the application to give you a random option from what you've typed. It can also completely delete the file where these options are stored.

Please make sure you run the application with admin priviligies.
